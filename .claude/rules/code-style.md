# コードスタイル

## HTML

- `lang="ja"` を設定
- `<meta charset="UTF-8">` と `<meta name="viewport" ...>` を必ず含める
- CSS・JSはHTMLに埋め込み（外部ファイル分割しない）
- セマンティックなHTML構造

## CSS

- CSS変数（カスタムプロパティ）を活用して一貫したテーマを維持
- モバイルファースト、レスポンシブ対応必須
- フォントはGoogle Fonts等からCDN読み込み。汎用フォントに頼らない
- ダークテーマをデフォルトとする（コンテンツの性質に合わない場合は例外OK）

## JavaScript

- モダンJS（ES2020+）を使用。トランスパイル不要な範囲で
- `const` / `let` を使用、`var` は禁止
- async/await を使用（.then チェーンは避ける）
- 外部ライブラリはCDNのscriptタグで読み込み

## SPA での注意

- GitHub Pages はクライアントサイドルーティングのリロードで404になる
- ハッシュルーター（`#/path` 形式）を推奨
- 必要に応じて 404.html にリダイレクトスクリプトを配置
